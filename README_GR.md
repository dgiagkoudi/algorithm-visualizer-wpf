# Algorithm Visualizer

Διαδραστική desktop εφαρμογή σε WPF και C# για την οπτικοποίηση βασικών αλγορίθμων ταξινόμησης, αναζήτησης και διάσχισης γράφων μέσω real-time animations και interactive interface.

## Λειτουργίες

- Οπτικοποίηση αλγορίθμων ταξινόμησης:
  - Bubble Sort
  - Merge Sort
  - Quick Sort
- Οπτικοποίηση αλγορίθμων αναζήτησης:
  - Binary Search
- Οπτικοποίηση διάσχισης γράφων:
  - Depth-First Search (DFS)
  - Breadth-First Search (BFS)
- Real-time animation εκτέλεσης
- Ρυθμιζόμενη ταχύτητα animation
- Δυναμική επιλογή μεγέθους πίνακα και γράφου
- Interactive controls:
  - Start
  - Pause
  - Stop
- Αριθμημένη σειρά επίσκεψης κόμβων
- Γραφική απεικόνιση πινάκων και γράφων

## Τεχνολογίες

- C#
- WPF (Windows Presentation Foundation)
- .NET Framework 4.8

## Τοπική Εκτέλεση

1. Κλωνοποιήστε το repository
```bash
git clone https://github.com/dgiagkoudi/algorithm-visualizer-wpf.git

cd algorithm-visualizer-wpf
```
2. Ανοίξτε το AlgorithmVisualizer.sln στο Visual Studio
3. Κάντε Build το solution
4. Πατήστε `F5` ή `Start` για να εκτελεστεί η εφαρμογή

## Λειτουργικότητα

Η εφαρμογή επιτρέπει στους χρήστες να:
- Παρακολουθούν βήμα-βήμα αλγορίθμους ταξινόμησης
- Βλέπουν traversal γράφων σε πραγματικό χρόνο
- Ρυθμίζουν δυναμικά την ταχύτητα εκτέλεσης
- Κάνουν Pause και Resume της εκτέλεσης
- Δοκιμάζουν διαφορετικά μεγέθη πινάκων και γράφων
- Κατανοούν τη λειτουργία των αλγορίθμων μέσω animations

## Εκπαιδευτικός Σκοπός

Το project δημιουργήθηκε ως εκπαιδευτικό εργαλείο για την κατανόηση του τρόπου λειτουργίας κλασικών αλγορίθμων μέσω οπτικοποίησης και αλληλεπίδρασης.

Μπορεί να χρησιμοποιηθεί για:
- Εκμάθηση αλγορίθμων και δομών δεδομένων
