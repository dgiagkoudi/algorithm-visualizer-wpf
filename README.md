# Algorithm Visualizer
🇬🇷 Read this in [Ελληνικά](README_GR.md)

An interactive desktop application built with WPF and C# for visualizing fundamental sorting, searching, and graph traversal algorithms through real-time animations and an interactive interface.

## Features

- Visualization of sorting algorithms:
  - Bubble Sort
  - Merge Sort
  - Quick Sort
- Visualization of searching algorithms:
  - Binary Search
- Graph traversal visualization:
  - Depth-First Search (DFS)
  - Breadth-First Search (BFS)
- Real-time animated execution
- Adjustable animation speed
- Dynamic array and graph size configuration
- Interactive controls:
  - Start
  - Pause
  - Stop
- Numbered node visitation order for graph algorithms
- Visual representation of arrays and graph structures

## Tech Stack

- C#
- WPF (Windows Presentation Foundation)
- .NET Framework

## Local Setup

1. Clone repository
```bash
git clone https://github.com/dgiagkoudi/algorithm-visualizer-wpf.git

cd algorithm-visualizer-wpf
```
2. Open AlgorithmVisualizer.sln in Visual Studio
3. Build the solution
4. Press `F5` or click `Start` to run the application

## Functionality

The application allows users to:

- Visualize sorting algorithms step-by-step
- Observe graph traversal in real time
- Control execution speed dynamically
- Pause and resume algorithm execution
- Experiment with different array sizes and graph configurations
- Learn how algorithms operate through animations

## Educational Purpose

This project was created as an educational tool for understanding how classic algorithms work internally through visualization and interaction.

It can be useful for:

- Students learning algorithms and data structures
- Educational demonstrations
- Understanding algorithm complexity and behavior
