# Algorithm Visualizer
Μια εφαρμογή WPF σε C# για την οπτικοποίηση βασικών αλγορίθμων ταξινόμησης και αναζήτησης με animation και interactive interface.

## Περιγραφή
Η εφαρμογή επιτρέπει στους χρήστες να δουν βήμα-βήμα πώς λειτουργούν οι αλγόριθμοι. Υποστηρίζει Bubble Sort, Merge Sort, Quick Sort, Binary Search, DFS και BFS, χρησιμοποιεί γραφικά για πίνακες και γράφους και επιτρέπει την επιλογή μεγέθους πίνακα/γράφου και ταχύτητας animation.  

## Χαρακτηριστικά

- Οπτικοποίηση αλγορίθμων ταξινόμησης (Bubble Sort, Merge Sort, Quick Sort).  
- Οπτικοποίηση αλγορίθμων αναζήτησης (Binary Search).  
- Οπτικοποίηση επίσκεψης κόμβων γράφου (DFS, BFS) με αριθμημένη σειρά επισκέψεων.  
- Έλεγχος μεγέθους πίνακα / αριθμού κόμβων.  
- Ρυθμιζόμενη ταχύτητα animation.  
- Start, Pause και Stop για τον έλεγχο της εκτέλεσης.

## Τοπική εκτέλεση

1. Κάνε clone το repository.
2. Άνοιξε το project στο Visual Studio.
3. Κάνε Build το project (```AlgorithmVisualizer.sln```).
4. Πάτησε Start (ή F5) για να τρέξει η εφαρμογή.
